# neurographrag

> Local GraphRAG memory for LLMs in a single SQLite file — zero external services required.

## Status

Work in progress — Phase 1+2 implementation.

## Quick Start

```bash
neurographrag init
neurographrag remember --name my-memory --type user --description "desc" --body "content"
neurographrag recall "search query"
```
